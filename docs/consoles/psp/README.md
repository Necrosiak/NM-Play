# PlayStation Portable — NM-Play Setup

## Requirements
- PSP with CFW (PRO-C, LME, or Infinity)
- WiFi adapter compatible with XLink Handheld Assistant (XLHA) — OR —
- Use PPSSPP emulator instead

## Setup — Real PSP
1. Install the **Kethen/aemu** Pro Online plugin on your PSP
   - Download from: https://github.com/Kethen/aemu/releases
   - Follow the install guide on the release page
2. Set the adhoc server in plugin config to: `networkmemories.com`
3. Set port to `27312`
4. Launch NM-Play on PC, select **PSP**, click **Connect**
5. Launch a game with Ad-Hoc multiplayer

## Compatible games (360+ titles)
Monster Hunter Freedom Unite, God of War Ghost of Sparta, Tekken 6,
Ridge Racer, Wipeout Pure, Call of Duty Roads to Victory, and many more.

## Notes
- PPSSPP and real PSP can play together in the same lobby
- Port offset must match between all players (default: 10000)

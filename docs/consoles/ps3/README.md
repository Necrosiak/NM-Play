# PlayStation 3 — NM-Play Setup

## Requirements
- PS3 (CFW or HEN recommended)
- Ethernet cable connected to same router as PC running NM-Play
- NM-Play desktop client

## Setup
1. Connect PS3 via ethernet to your router
2. Launch NM-Play on your PC, select **PlayStation 3**, click **Connect**
3. Launch a LAN-compatible game

## Compatible games
Warhawk, Killzone 2/3, Gran Turismo 5/6, Call of Duty series,
Resistance series, and more.

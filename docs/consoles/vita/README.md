# PlayStation Vita — NM-Play Setup

## Requirements
- PS Vita with CFW (Ensō or HENkaku)
- Kethen/aemu plugin installed
- WiFi connection on same network as PC running NM-Play

## Setup
1. Install the **Kethen/aemu** plugin on your Vita
   - Download from: https://github.com/Kethen/aemu/releases
2. Set adhoc server to: `networkmemories.com:27312`
3. Launch NM-Play on PC, select **PS Vita**, click **Connect**
4. Launch a game with Ad-Hoc multiplayer

## Compatible games (155+ titles)
Monster Hunter series, Persona 4 Golden, Soul Sacrifice, Freedom Wars, and more.

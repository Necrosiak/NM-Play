# Wii — NM-Play Setup

## Requirements
- Wii with Homebrew Channel
- USB Loader GX or similar
- Ethernet adapter (official or third-party)
- NM-Play desktop client

## Setup
1. Connect Wii via ethernet to your router
2. Launch NM-Play on your PC, select **Wii**, click **Connect**
3. Launch Mario Kart Wii (with LAN patch) via USB Loader

## Compatible games
Mario Kart Wii (with patch), limited other titles via homebrew mods.

## Notes
- No official LAN games exist on Wii — homebrew/patches required
- Wii and Dolphin emulator players can play together

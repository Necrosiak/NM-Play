# Xbox 360 — NM-Play Setup

## Requirements
- Xbox 360 (RGH/JTAG recommended)
- Ethernet cable connected to same router as PC running NM-Play
- NM-Play desktop client

## Setup
1. Connect Xbox 360 via ethernet to your router
2. Launch NM-Play on your PC, select **Xbox 360**, click **Connect**
3. Launch a System Link compatible game

## Compatible games
Halo 3, Halo Reach, Halo 4, Gears of War series, Call of Duty series,
Forza Motorsport series, and many more.

# GameCube — NM-Play Setup

## Requirements
- GameCube with **Broadband Adapter (BBA)**
- Ethernet cable connected to same router as PC running NM-Play
- NM-Play desktop client

## Setup
1. Connect GameCube BBA via ethernet to your router
2. Launch NM-Play on your PC, select **GameCube**, click **Connect**
3. Launch a LAN-compatible game on GameCube

## Compatible games
Mario Kart: Double Dash!! (up to 16 players / 4 GameCubes),
Kirby Air Ride, 1080 Avalanche, and others.

## Notes
- Mario Kart: Double Dash!! supports up to 4 GameCubes × 4 players = 16 players total
- GameCube and Dolphin emulator players can play together

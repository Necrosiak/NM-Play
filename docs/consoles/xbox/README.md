# Xbox (Original) — NM-Play Setup

## Requirements
- Original Xbox (modded recommended for full compatibility)
- Ethernet cable connected to same router as PC running NM-Play
- NM-Play desktop client

## Setup
1. Connect Xbox via ethernet to your router
2. Launch NM-Play on your PC, select **Xbox**, click **Connect**
3. Launch a System Link compatible game

## Compatible games
Halo 2 (up to 16 players), Halo CE, Star Wars Battlefront 1/2,
Call of Duty 2, Burnout 3, and 200+ more System Link titles.

## Notes
- Xbox and Xemu emulator players can play together
- For online revival (Xbox Live replacement), see Insignia: https://insignia.live

# Xenia (Xbox 360) — NM-Play Setup

## Requirements
- Xenia Canary Netplay build
- NM-Play desktop client

## Setup
1. Download Xenia Canary Netplay from: https://github.com/AdrianCassar/xenia-canary
2. Launch NM-Play, select Xbox 360, click Connect
3. In Xenia config: set network_mode to System Link
4. Launch a System Link compatible game

## Notes
- Xenia and real Xbox 360 players can play together
- Use Xenia Canary Netplay build specifically — not standard Xenia

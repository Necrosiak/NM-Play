# Dolphin (GameCube/Wii) — NM-Play Setup

## Requirements
- Dolphin emulator (latest stable or dev build)
- NM-Play desktop client

## Setup
1. Launch NM-Play, select GameCube or Wii, click Connect
   NM-Play will auto-configure Dolphin network settings
2. In Dolphin: Config → GameCube → SP1 → set to Broadband Adapter (BBA)
3. Launch a LAN-compatible game

## Manual network config
- Dolphin → Config → Network
- Enable Broadband Adapter
- NM-Play handles IP assignment automatically

## Notes
- Dolphin and real GameCube/Wii players can play together in same lobby
- For GameCube: BBA required on real hardware
- Recommended: Dolphin dev build for best LAN compatibility

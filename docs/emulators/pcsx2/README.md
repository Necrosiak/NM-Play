# PCSX2 (PlayStation 2) — NM-Play Setup

## Requirements
- PCSX2 (latest nightly recommended)
- NM-Play desktop client
- Npcap installed (Windows)

## Setup
1. Launch NM-Play, select PlayStation 2, click Connect
   NM-Play auto-configures PCSX2 network adapter
2. In PCSX2: Config → Network & HDD → Ethernet → enable, set adapter

## Manual config
- Ethernet Device: select your physical network adapter or TAP
- NM-Play creates a virtual TAP adapter automatically

## Notes
- PCSX2 and real PS2 players can play together
- Some games require specific network configs — check game compatibility

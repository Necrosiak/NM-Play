# Xemu (Xbox Original) — NM-Play Setup

## Requirements
- Xemu emulator (latest build)
- NM-Play desktop client
- Npcap installed (Windows)

## Setup
1. Launch NM-Play, select Xbox, click Connect
2. In Xemu: Machine → Settings → Network
   - Enable networking
   - Backend: NAT or PCap
   - NM-Play handles the rest automatically

## Notes
- Xemu and real Xbox players can play together
- PCap backend gives best compatibility for System Link games
- Halo 2 is the most popular System Link game

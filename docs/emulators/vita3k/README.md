# Vita3k (PlayStation Vita) — NM-Play Setup

## Requirements
- Vita3k emulator (latest build)
- NM-Play desktop client

## Setup
1. Launch NM-Play, select PS Vita, click Connect
2. In Vita3k: Settings → Network
   - Enable Ad-Hoc
   - Server: `networkmemories.com`
   - Port: `27312`
3. Launch a game with Ad-Hoc multiplayer

## Notes
- Vita3k and real PS Vita (with Kethen/aemu) can play together
- Ad-Hoc support in Vita3k is experimental

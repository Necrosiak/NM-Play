# RPCS3 (PlayStation 3) — NM-Play Setup

## Requirements
- RPCS3 (latest build)
- NM-Play desktop client

## Setup
1. Launch NM-Play, select PlayStation 3, click Connect
2. In RPCS3: Config → Network
   - Network Status: Connected
   - Network Handler: Standard/Rollback
3. Launch a LAN-compatible game

## Notes
- RPCS3 and real PS3 players can play together in same lobby
- Internet connection required even for LAN emulation

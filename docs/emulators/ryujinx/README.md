# Ryujinx / Citron (Nintendo Switch) — NM-Play Setup

## Option 1 — Via NM_ldn_mitm_relay (Recommended)

Ryujinx supports ldn_mitm mode natively.

1. In Ryujinx: Options → Settings → Multiplayer
   - Mode: ldn_mitm
   - Disable Guest Internet Access
2. NM-Play connects Ryujinx to the NetworkMemories relay automatically
3. Ryujinx and real Switch (CFW) players can play together

## Option 2 — NM-Play client (LAN mode games only)

For games with native LAN mode:
1. Launch NM-Play, select Nintendo Switch, click Connect
2. Launch game in LAN mode

## Notes
- ldn_mitm mode supports ALL local wireless games (198+ titles)
- LAN mode supports only the 25 games with native LAN support
